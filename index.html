Vue Router Tab

基于 Vue Router 的路由页签组件。

快速上手 →

简洁至上

配置简单，引入组件后通过路由的 "meta" 元信息配置页签信息即可运行。

规则定制

方便定制页签规则。除了默认的规则，您还可以全局和精准控制每个路由的页签打开方式。

Vue Router 驱动

页签响应 "Vue Router" 路由，您可以使用 "Vue Router" 提供的强大功能。
